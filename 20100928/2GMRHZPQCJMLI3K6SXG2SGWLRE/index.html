煮題　台灣麪滑早餐
罐頭紅豆茸代替

年糕可以用吃剩或隔夜紅豆沙的糖水來做，亦都可用做糕點的罐頭紅豆茸代替，但就要小心控制甜度；另外，如果年糕擺得耐變硬，可放入微波爐翻熱20至30秒，立即變回軟滑。

加白醋半凝固

鹹豆漿中加入白醋，可令豆漿稍為凝固，這樣才好吃，所以白醋是不能缺少的調味料。

先煮滾水免燶

米漿中含澱粉質，容易沉澱，應先在鍋中煮滾水，才注入米漿，其間不停攪動，才可防止黐底。

烤紅豆年糕

01hr00min
製作難度：★★☆☆☆
份量：4人

紅豆年糕外皮焗得鬆脆，入面相當煙韌，富有豆香，但味道不會過甜，做早餐非常飽肚。

材料

糯米粉230克、雞蛋2隻、生油80毫升、砂糖50克、紅豆糖水連豆250毫升、牛奶80毫升、呍呢拿油少許

做法

1. 雞蛋拂勻，注入牛奶、生油、糯米粉、紅豆糖水、砂糖、呍呢拿油拌勻成粉漿。

2. 盤上噴灑生油，灑入少許糯米粉，篩勻後將多餘的糯米粉倒出。注入紅豆粉漿後，放入已預熱180℃焗爐焗20分鐘。

3. 將年糕取出，蓋上錫紙，再放入焗爐，以180℃焗10分鐘後，將錫紙取走再放入焗爐以180℃焗10分鐘，用牙籤插入年糕測試，如果不沾黏即表示熟透。

4. 將紅豆年糕取出，放凉後切塊即成。

鹹豆漿

00hr50min
製作難度：★★☆☆☆
份量：4人

鹹豆漿材料好豐富，油條、蝦皮全部炒得香口而不油膩，豆漿又滑又綿，口感一流。

材料

無糖原味豆漿600毫升、榨菜（切碎）100克、五香豆乾（切丁粒）1塊、蝦皮/葱花各10克、油條1/4條

調味料︰白醋1茶匙、麻油1/4茶匙、生抽/鹽各少許

做法

1. 將油條切薄片；燒熱鑊下少許油，放入其中一半油條薄片，以大火炒至脆身，盛起備用。

2. 用原鑊放入蝦皮，以大火炒香，盛起備用；再放入豆乾粒以大火炒香，盛起備用；再放入榨菜粒以大火炒香，盛起備用。

3. 將豆漿注入鍋中用中火煮滾；將醋、鹽、生抽放入碗中，撞入已煮熱的豆漿。

4. 灑上炒好的豆腐乾粒、榨菜粒、蝦皮、油條，再撒入其餘油條薄片、葱花，最後灑少許麻油調味，即成。

原味麪茶

00hr40min
製作難度：★★☆☆☆
份量：4人

台灣人用麪茶代替牛奶和糊仔，給小朋友吃，做法簡單但營養十足，伴油炸鬼食，味道好夾！

材料

中筋麪粉100克、砂糖2湯匙、清水600毫升、熟白芝麻/熟杏仁片/熟杏仁碎各適量

做法

1. 麪粉放入鑊，以文火炒約3至5分鐘至淺啡色，並且散發出香味，加入砂糖炒勻。

2. 將炒熟的麪粉盛入碗中，再放入杏仁片、杏仁碎與白芝麻。

3. 將清水以大火煮沸後，再逐少注入熱滾水於（2），一邊拌勻至麪粉溶化，即成。

麪茶可加入黑芝麻、杏仁片來冲泡，另有一番風味。

花生米漿

01hr00min
製作難度：★★★☆☆
份量：4人

米漿質地濃稠綿滑，而且好有花生香氣，比花生糊味道更加清甜！

材料

珍珠米/砂糖各100克、花生150克、水2公升、花生醬3湯匙

做法

1. 花生倒入淺碟，加少許清水略浸，放入微波爐加熱3分鐘，取出拌勻，再加熱1分鐘至熟透，用廚紙包裹，將花生衣擦去，放凉備用。

2. 珍珠米加入花生中，注入800毫升清水浸泡約3小時。

3. 珍珠米連同花生及浸米水，一同注入攪拌機中，打勻成米漿，再加入花生醬打勻。

4. 將其餘1,200毫升清水於鍋中以大火煮滾，注入打好的花生米漿拌煮，滾後轉小火，一邊攪拌一邊加砂糖，煮約5分鐘至砂糖溶化，即成。
