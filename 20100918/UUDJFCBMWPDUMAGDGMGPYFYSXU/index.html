卡通便當　小丸子木魚包包面

小朋友正值發育階段，如果偏食或唔食飯，確實令家長好頭痛，倒是日本的媽媽有本事，想到將日日食到悶的飯糰搓成不同的卡通人物，又將平凡的配菜，變成花朵般漂亮，色彩繽紛就算大人見到一樣會流口水！

記者︰黃家欣
攝影︰楊錦文

Profile
田中佳代子（烹飪導師）

移居香港逾十年的日本人，擅長炮製家庭風味的日本菜式，有着日本人的精巧細緻手工。

鮮艷伴菜

田中太太指日本媽媽的飯盒哲學是，將不同顏色的食物放入飯盒，除可引起小朋友食慾外，還代表營養均衡，所以伴菜的手工亦很講究。

甘笋片︰

用曲奇餅模吸出形狀，可先吸形後才以滾水灼熟，因生甘笋較爽，切口會更整齊。

腸仔花︰

將頭及尾約2厘米的腸仔切出，米字後用滾水灼熟，中間放粟米或枝豆即可。

蘋果花︰

先切薄片，於蘋果皮上交叉，用刀角對着蘋果皮下半厘米下，即做出三角形圖案。

蛋卷︰

用鹽水將菠菜灼熟，放凉後榨乾。燒熱鑊下油，加蛋漿後放菠菜並捲起，切片即成。

火腿花︰

將火腿對摺，於背部切出數條平行斜線，然後捲起，用牙籤固定底部。

小丸子漢堡扒

01hr00min
製作難度：★★★☆☆
份量：2盒

漢堡扒混入牛肉與豬肉，食落肉味更香、更濃，煎香後加上濃濃的醬汁，更滋味。

材料

壽司飯2碗、木魚碎少許、紫菜/火腿各1片、免治牛肉70克、免治豬肉30克、雞蛋1隻、洋葱1/3個、麪包糠/魚肉色粉/牛油各少許

調味料︰醬油/鹽/糖各1茶匙、胡椒粉少許、茄汁１1/2湯匙、OK汁/味醂各1湯匙

做法

1. 木魚碎加醬油拌勻；飯一開二，平放於保鮮紙上，加少許已調味的木魚碎，包好搓成圓形。

2. 紫菜剪成頭髮、眼睛及眼眉；用火腿剪成嘴巴，放在飯糰。在面珠位撒上魚肉色粉，備用。

3. 洋葱切碎，放熱鑊加牛油炒香，以胡椒粉、鹽調味，盛起放凉備用。

4. 雞蛋拂勻拌入麪包糠後，加牛肉、豬肉、洋葱碎，用手攪勻備用；將茄汁、OK汁、糖、味醂調勻成為醬汁。

5. 燒熱鑊放少許油，將肉搓成迷你肉餅放入鑊，以中火煎至熟透金黃，轉慢火加入醬汁，略為拌勻即可盛起，與飯糰及伴菜放入飯盒中。

蠟筆小新炸雞

00hr50min
製作難度：★★☆☆☆
份量：2盒

小新造型鬼馬，混入三文魚碎，顏色逼真外，食落亦帶鹹香，加上無骨的炸雞腿肉，小朋友食得更開心！

材料

壽司飯2碗、三文魚碎10克、雞髀肉2件、火腿/紫菜各1片、芝士1/2片、生意粉1條、雞蛋1隻

調味料︰蒜鹽1茶匙、麪粉１2/3湯匙、胡椒粉少許、醬油2湯匙

做法

1.飯拌入三文魚碎後分成兩大份及四小份。將大份的趁熱各以保鮮紙包好，搓成上窄下闊的面形；將四小份各用保鮮紙包起，搓成橢圓形成耳仔。用生意粉將面及耳仔穿起。

2. 紫菜剪成頭髮、眼睛及眼眉，放在飯糰上；將火腿剪成橢圓的嘴巴放於飯糰；用飲管在芝士上吸出小圓形，放上紫菜成為眼珠。

3. 雞髀肉一開二備用；將雞蛋拂勻，加入蒜鹽、麪粉、胡椒粉、醬油拌勻，放雞肉沾均勻。

4. 用大火燒滾半鍋油，放入雞件炸至金黃熟透，撈起瀝油後，即可與伴菜及飯糰排入飯盒中。

多啦A夢雞堡

01hr10min
製作難度：★★★☆☆
份量：2盒

雞肉漢煲加入豆腐來搓，入口特別鬆軟滑溜，蘸醬汁特別香甜，小朋友一定啱食！

材料

壽司飯2碗、紫菜/火腿1片、芝士1/2片、即食魚肉腸1/2條、免治雞肉100克、豆腐1/2磚、麪包糠1湯匙、雞蛋1隻、洋葱（切碎）1/3個

調味料︰鹽1/2茶匙、胡椒粉少許、醬油適量、味醂2湯匙、糖1茶匙

做法

1. 將飯一開二，趁熱放入保鮮紙搓成圓形，紫菜剪出半月形、嘴巴及鬚放於飯糰上；芝士吸出圓形眼睛，再放以紫菜剪成的圓形眼珠；魚肉腸切成薄片變成鼻子，備用。

2. 雞蛋拂勻後加入麪包糠、洋葱拌勻，以鹽、胡椒粉、醬油調味後，加入雞肉及豆腐拌勻；將醬油、味醂、糖拌勻成醬汁，備用。

3. 將雞肉搓成迷你肉餅，放入熱鑊加少許油，以中火煎至金黃色，轉慢火加入醬汁煎至入味，即可盛起，與飯糰及伴菜一同放入飯盒。

麪包超人牛肉卷

00hr40min
製作難度：★★☆☆☆
份量：2盒

麪包超人飯糰混入木魚碎更為香口，而包着秋葵的牛肉卷，爽甜又具肉香。

材料

壽司飯2碗、木魚碎少許、牛肉薄片3片、火腿/紫菜各1片、即食魚肉腸1/2條、秋葵4條　

調味料︰醬油少許、鹽1茶匙

做法

1. 醬油與木魚碎拌勻後，加入壽司飯用飯勺拌勻，一分為二，趁熱用保鮮紙包好，搓成圓形。

2. 將魚肉腸切薄片成面珠；火腿剪出大橢圓形成鼻子；紫菜剪出眼睛及嘴巴，放於飯糰上，備用。

3. 煮滾水，加1茶匙鹽，放入秋葵灼熟，撈起捲上牛肉薄片。鑊加油將牛肉卷以中火煎至金黃，即可盛起切段。

4. 將牛肉卷、飯糰及伴菜排入飯盒中，即成。
