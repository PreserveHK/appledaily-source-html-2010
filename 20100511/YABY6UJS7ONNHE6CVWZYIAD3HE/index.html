煮題　沖繩鄉土料理

魚肉要先拍扁，增加其黏性，做好的吉士魚蛋也較彈牙。

家中沒有焗爐，也可用蒸的方法，然後將吉士魚蛋切件來煎，味道也很好。

苦瓜的苦味來自白色的瓤，要把核和瓤挖乾淨，才不會苦。

吉士魚蛋

00hr50min
製作難度：★★★☆☆
份量：4人

吉士魚蛋是用白身魚茸加蛋去做的香滑魚蛋，因為似吉士蛋糕，所以叫吉士魚蛋，是沖繩人日常小食。因為本身金金黃黃，亦是節日必備配菜之一。

材料

鯰魚柳200克、鯛魚柳50克、蛋5隻、青芥辣少許

調味：砂糖20克、鹽1/2茶匙、味醂1茶匙

做法

1. 先將鯰魚柳與鯛魚柳用菜刀拍扁，增加黏度。將兩款魚肉混在一起，用刀背剁碎。

2. 將魚肉放入大碗內，雞蛋逐隻打入，跟魚肉混合，再加另一隻，直至攪勻成為魚漿。

3. 然後加砂糖、鹽、味醂再攪勻，倒入已掃油的焗模內，掃平表面，輕輕拍打一下，讓空氣擠出，焗出來會結實一點。在焗盤內注入約5毫米高的熱水。

4. 焗爐預熱至180℃，放入魚漿焗25分鐘左右，用竹籤插一下，沒有黏液、一穿到尾即成。

5. 然後將魚蛋放凉切片。

6. 燒熱鑊加油，以中火煎魚蛋至金黃，吃時伴以青芥辣即成。

黑糖香蕉蛋糕

01hr00min
製作難度：★★★☆☆
份量：4人

沖繩因為有美軍駐守，加上盛產香蕉，所以有人引入香蕉蛋糕的做法，加上沖繩著名的黑糖，就成為沖繩名物之一。

材料

無糖班戟粉400克、香蕉4隻、蛋4隻、牛奶100毫升、牛油100克（室溫放軟）、黑糖4湯匙

做法

1. 牛油打軟，加黑糖打勻備用。

2. 蛋拂勻，加班戟粉打成漿，加牛奶及混好的牛油攪勻。

3. 香蕉去皮取肉壓爛。然後將香蕉肉加入麪糊中攪勻至滑身。

4. 將麪糊放入已掃油的焗盤內，放入已預熱到180℃的焗爐焗40分鐘左右即成。

苦瓜炒雜錦

00hr30min
製作難度：★★★☆☆
份量：2人

沖繩人愛用美國引進的午餐肉加蛋及蔬菜做炒雜錦，苦瓜是沖繩代表蔬菜，苦瓜炒雜錦成為沖繩代表菜式。

材料

沖繩苦瓜1個、木棉豆腐200克、雞蛋1個、煙腩肉100克、木魚花適量

調味料：鹽/糖/日本醬油各適量

做法

1. 苦瓜切開兩半，挖芯，切薄片。

2. 煙腩肉切小塊。豆腐切小塊。

3. 燒熱鑊放入適量油，倒入苦瓜轉中火炒勻，想腍身一點可灒少許水一起炒，然後倒入豆腐、煙腩肉炒勻。

4. 撒鹽、糖略炒，加醬油提味炒勻。

5. 倒入打散的雞蛋炒勻，上碟後撒上木魚花即成。

沖繩風豚之角煮

01hr80min
製作難度：★★★☆☆
份量：4人

因為沖繩近台灣，其實料理風格都受中國影響，這個豚之角煮可說是沖繩版的東坡肉了。

材料

五花腩800克、昆布20厘米、昆布木魚素1茶匙、琉球泡盛酒1杯、沖繩黑糖/低鹽日本醬油各1/4杯、葱2條（切段）、薑4片

做法

1. 先將五花腩放入一鍋水中，然後開火煮至滾起，汆淨肥油及血水，洗淨備用。

2. 煲中加水、昆布和腩肉，再加入適量的水以蓋過腩肉面，先用大火煮滾，然後轉細火煮約45分鐘。

3. 取出腩肉放凉後裁齊邊位，再切成大件。撈起昆布。鍋中剩下湯汁，加入腩肉、薑片、葱段、沖繩黑糖、低鹽醬油、泡盛酒、昆布木魚素等調味料用大火煮滾，再煮5分鐘。

4. 然後轉小火煮1小時左右，至適合軟腍度及入味，上碟即可。
