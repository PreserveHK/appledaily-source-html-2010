無火氣　醉甜美

女士一向愛飲甜酒，甜甜的特別易入喉。BAILEYS除了味甜，還很香滑，除了淨飲，其幼滑忌廉質感更適合做甜品，萬麗海景酒店糕餅部主廚董遠華就用了這種酒為大家示範4款甜品，他說：「低溫烹煮忌廉酒，不讓酒精完全揮發，最適合做朱古力、咖啡等甜品。」

記者：黃潔蓮　
攝影：伍慶泉、梁細權
示範：萬麗海景酒店糕餅部主廚董遠華

董遠華（萬麗海景酒店糕餅部主廚）

任糕餅師近二十年，曾於富麗華酒店、國泰城、跑馬地餐廳任職，擅長烹調法式甜品，如朱古力CremeBrulee。

百變忌廉酒

愛爾蘭出產的BAILEYS，是以忌廉與威士忌為基調，調配出一種滑溜香甜的忌廉酒。有原味、焦糖味、薄荷朱古力味和新出品的咖啡味。除了淨飲加冰外，還可以調製成特色雞尾酒。

薄荷芝士餅

3hr15min
製作難度：★★☆☆☆
份量：4人

材料

黑朱古力蛋糕（3.5×11吋）2塊、咖啡糖水*50克、唂咕粉30克、雜莓100克

芝士餡料：忌廉400毫升、砂糖60克、蛋黃3隻、意大利軟芝士（Mascorpone）400克、魚膠片15克、薄荷朱古力味BAILEYS60毫升

*咖啡糖水：濃咖啡70毫升、砂糖16克、原味BAILEYS20毫升（餘下做其他甜品）

做法

1. 芝士餡：忌廉與半份砂糖打起；蛋黃與餘下的糖打起。魚膠片用冰水浸軟，加入煮滾的薄荷朱古力酒中煮至溶。然後將薄荷朱古力酒注入蛋黃攪勻，混合忌廉及軟芝士製成餡。

2. 咖啡糖水：將材料全放進煲中煮至糖溶，攤凉備用。

3. 將一塊朱古力蛋糕放入模內，掃上咖啡糖水，注入餡料至半滿；重複一次以上動作至餡料填滿，放雪櫃雪3小時至凝固。

4. 切成4件後撒上唂咕粉上碟，配上雜莓即成。

白朱古力慕絲

2hr15min
製作難度：★☆☆☆☆
份量：1人

材料

手指餅3條、咖啡糖水適量、芒果粒1/2杯、白朱古力碎適量

慕絲材料：原味BAILEYS10毫升、Kahlua咖啡酒10毫升、濃咖啡1.5毫升、魚膠片3.5克、糖霜15克、忌廉100克

做法

1. 慕絲：將忌廉與糖霜一起拂起至六成半。魚膠片放冰水中浸軟，加入煮滾的濃咖啡、原味BAILEYS與咖啡酒煮至溶，將之放入忌廉中攪勻。

2. 在杯中放入1/3杯慕絲，將手指餅切半，蘸上咖啡糖水，然後放在慕絲上，再蓋上一層慕絲，放浸咖啡糖水的手指餅，最後蓋上慕絲，掃平表面，放雪櫃雪2小時。

3. 取出慕絲，放上芒果粒及白朱古力碎即成。

軟芝士餅班戟

2hr30min
製作難度：★★★☆☆
份量：4人

遠看還以為是蛋卷，實質是捲起如壽司般的班戟卷，非常香滑。

材料

朱古力蛋糕（10×0.5吋）4塊、朱古力手指餅2條、綠茶雪糕1球、咖啡糖水適量

班戟：濃咖啡25毫升、高筋麪粉85克、砂糖9克、雞蛋1隻、沙拉油8毫升、奶100毫升

餡料：意大利軟芝士125克、砂糖32克、蛋黃1隻、原味BAILEYS15毫升、濃咖啡20毫升、魚膠片2.5克、忌廉（拂起）225克

做法

1. 班戟：將所有材料混合至有筋性。燒熱平底鑊，掃少許油，放入班戟漿煎成薄班戟，一共煎4塊，放凉備用。

2. 餡料：軟芝士、糖、蛋黃及原味BAILEYS攪勻。魚膠片浸冰水至軟，加入已煮滾的濃咖啡內攪拌至溶。將之倒入軟芝士漿中拌勻，再加入拂起的忌廉攪勻，放雪櫃雪1小時。

3. 將餡料抹在班戟上，預留1/4吋邊位，放上已浸咖啡糖水的朱古力蛋糕，然後捲起，放雪櫃雪1小時。

4. 取出切成4件放碟上，旁邊放綠茶雪糕及朱古力手指餅即成。

拿破崙餅

0hr30min
製作難度：★★★☆☆
份量：4人

市面上售賣拿破崙餅不多，找到好吃的拿破崙餅更是少上加少。酥皮要夠脆、蛋糕要夠軟綿、忌廉要夠香滑，才是最佳的拿破崙餅。

材料

焗好的急凍酥皮（1.5×6吋）3塊、糖霜20克、咖啡糖水50毫升

蛋糕：蛋白100克、砂糖80克、高筋麪粉25克、合桃碎38克

牛油忌廉：牛油50克、糖霜25克、原味BAILEYS20毫升

做法

1. 蛋糕：蛋白與砂糖拂至半起，加高筋麪粉、合桃碎拌勻，倒進長焗盤上掃平，放入已預熱至230℃的焗爐焗11分鐘，放凉後切成3塊1.5×6吋。

2. 牛油忌廉：將所有材料拂勻至淡黃色。

3. 在一塊酥皮上塗牛油忌廉，放上一片蛋糕片，塗上咖啡糖水。再加牛油忌廉、放蛋糕、掃咖啡糖水、塗牛油忌廉，最後是酥皮。

4. 將拿破崙餅切成4件，上碟後撒上糖霜即成。

使用高筋麪粉與蛋白做的蛋糕，會較煙韌。

ChocoB52

0hr10min
製作難度：★★★☆☆
份量：4人

這個是取B52的三種層次，配以咖啡酒、橙酒與忌廉酒混合，加上熱朱古力，是冬天最暖的熱酒。

材料

原味BAILEYS1.5安士、GrandMarnier橙酒/Kahlua咖啡酒各1/2安士、熱朱古力300毫升、棉花糖/朱古力碎各適量

做法

1. 先用熱水倒入玻璃杯弄熱杯身，倒掉。

2. 所有酒倒入杯中，撞入熱朱古力，最後撒上棉花糖及朱古力碎即成。
