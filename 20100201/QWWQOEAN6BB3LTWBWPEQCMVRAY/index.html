夠Car事：AstonMartinRapide　四翼齊飛

要揀今年紅爆車款，非房跑莫屬，唔好以為將原來兩門的跑車拉長，加後座再加兩扇門，咁就話係房跑。你諗吓，連Supercar英國車廠AstonMartin都加入四門房跑戰團，咁就知道房跑都分好多層次，花得起錢嘅當然要求更加高，水準當然又非一般。

記者：趙慧敏
攝影：陳盛臣

車門上翹

Rapide身為AstonMartin首架四門跑車，身價不菲之餘，車廠更係落足本錢心機，用「萬眾期待」，「千呼萬喚」形容車迷心情，可能有啲老土，但就絕不離事實。終於一輛左軚版Demo過境香港，大家有機會一睹其真面目。Rapide設計貫徹AstonMartin超級跑車風格，低重心及流線型。但最令人期待的，還是車廠如何將車身加上兩扇門及後座，仍保留其跑味。首先四門採用天鵝翼設計，車門開啟後微微向上翹起，令她在一般跑車群中更為突出。至於鴨尾型的定風翼及無框車窗設計等，都令全車線條結構更加融和。至於車櫳，豪華感自然不能少，人手製四座獨立跑車座椅，高貴之餘亦散發跑味，流線中控台及鋼琴面裝飾，似豪房多於豪跑，不過由中控台一直伸展至後座的獨立空調，則佔據不少空間，無論前後座都稍嫌狹窄。

1. 車門天鵝翼設計，微微上翹。

2. 頭燈設計為焦點所在，醒目而有層次感。

3. 20吋胎軨，配合房跑格調。

4. AstonMartin一貫鯊魚腮設計。

5. 引擎裝上頂Bar，睇得又劈得。

雙物料式制動

對於AstonMartin在跑車上的技術，車迷毋庸置疑。Rapide屬前置引擎，後輪驅動，採用人手打造的6公升V12引擎，擁有470匹馬力，0-100km/hr只需5.3秒，最高時速達303公里。為求車身增大後，亦可保持速度，Rapide用上較輕物料，車重只有1,950kg，用鋁合金車架。相對於DB9，Rapide車長只稍增30厘米，而全新設計的油缸容量則達90.5公升，令Rapide足以連續行駛300公里仍能保持動力。此外，車廠亦放入不少嶄新元素，Rapide是AstonMartin首次採用雙物料式制動系統車款，煞車碟以鑄鐵及鋁製，比一般鑄鐵煞車系統輕了15-20%，而全新的水力制動協助系統（HBA），能夠於緊急情況下縮短煞車距離。此外，還用上可調式避震系統（ADS），可預先設定5種不同情況，駕駛時懸掛設定便可自動調節，六前速Touchtronic2加減波箱，亦令轉檔的反應更快更暢順。

1. 跑車桶座有跑味有貴氣，但腳部伸展位置稍嫌不足。

2. 真皮軚環連加減波撥片。

3. 中控台配合鋼琴面，豪華感非同一般。

4. 流線的外形，凸顯肌肉感。
