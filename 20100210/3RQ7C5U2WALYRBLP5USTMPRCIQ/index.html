煮題　富貴團年菜
選一斤四隻田雞

田雞不用揀大隻，因太大肉質略粗糙，細隻的又嫌無肉，選一斤有四隻的最適合。

1.生財顯貴雞

01hr00min
製作難度：★★★☆☆
份量：6人

現在物資豐富，經常有機會吃到，每到過時過節，吃雞更成為習慣，吃厭了一般白切雞，不妨試做這款加了蜆蚧醬的雞，味濃又香。

材料

雞1隻、蒜茸1湯匙、鹽/糖/雞粉各1茶匙、蜆蚧醬114克、生抽1湯匙、薑/乾葱各76克、葱粒1湯匙、生菜380克、紹酒38克、紅椒絲/葱絲各少許　

汁料：紹酒/葱段各38克、雞湯1碗、雞粉1茶匙、冰糖19克、蠔油/生粉芡各1湯匙、薑2片、乾葱/蒜茸各1茶匙、陳皮絲5克

做法

1. 雞塗上生抽，燒熱鑊加油放入雞略煎後盛起。

2. 燒熱鑊下薑、乾葱、蒜茸、葱粒、半份蜆蚧醬、紹酒、鹽、糖及雞粉炒勻，盛起後塞入雞櫳內醃30分鐘。

3. 另燒熱鑊加油，用中火爆香薑片，加葱段、剩餘蜆蚧醬、紹酒及雞湯，轉大火煮滾，加雞粉、冰糖及蠔油煮滾，再放入雞，冚蓋用中火煮23分鐘。

4. 燒滾水加少許油、鹽及糖，放入生菜灼熟上碟。

5. 雞取出，隔起煮雞汁。燒熱鑊下油，爆香乾葱、蒜茸、陳皮絲及已隔渣煮雞汁，開大火煮滾，埋生粉水芡。

6. 雞稍攤凉斬件上碟，淋上醬汁，放葱絲及紅椒絲即成。

2.大展鴻圖（香葱醬油田雞腿）

0hr30min
製作難度：★★☆☆☆
份量：4人

撐起的田雞腿有大展拳腳、開枝散葉之意，加上鮮味醬汁，特別惹味。

材料

田雞800克、乾葱茸/蒜茸/薑茸各5克、麪粉19克、葱花3克、紅椒絲少許　

醃料：葱2棵（切段）、薑粒10克、鹽1/2茶匙、糖1茶匙、雞粉/胡椒粉各少許、紹酒1茶匙、麻油2克　

調味料：醬油/糖各8克、老抽/麻油各2克、蠔油/雞粉各5克、紹酒1湯匙、胡椒粉1克、清湯70毫升

做法

1. 將田雞起骨，開兩邊起出脊骨，再開大髀位，將大腿骨拉出，斬去骨頭，然後加薑粒、葱段、鹽、糖、雞粉、紹酒、胡椒粉及麻油略醃。

2. 將田雞拍上麪粉，燒熱鑊加油，放入田雞用中火以半煎炸方式將田雞煎熟，盛起上碟。

3. 將清湯、雞粉、蠔油、醬油、糖、紹酒、麻油及胡椒粉拌勻成芡汁。

4. 燒熱鑊加油，用中火爆香薑茸、蒜茸及乾葱茸，加芡汁煮滾，放入田雞炒勻，加老抽調味，再加葱花及紅椒絲炒勻後上碟即成。

3.幸福團圓（花旗參燕窩湯圓）

0hr30min
製作難度：★☆☆☆☆
份量：4人

湯圓又叫元宵，相傳袁世凱認為「袁消」一名不吉利，所以改名為湯圓。師傅示範的用了燕窩及花旗參，味道清新。

材料

熱水608克、冰糖152克、花旗參114克　

餡料：砂糖76克、魚膠粉15.2克、清水304克、已發燕窩100克　

皮料：糯米粉304克、生油38克、清水342克

做法

1. 先做餡料。魚膠粉加水拌勻放入煲煮滾，加砂糖煮至溶，倒入碟待5分鐘，鋪上燕窩，放入雪櫃雪至凝固。

2. 將糯米粉逐少加水搓勻，再加生油搓成粉糰。

3. 將粉糰搓長分成10克一粒，搓圓壓平，包入餡料再搓圓，放入雪櫃雪10分鐘至定形。

4. 將花旗參加半份熱水入碗焗15分鐘。將剩餘的熱水放入煲，加冰糖煮至溶，再倒入花旗參水，隔渣煮滾後盛起。

5. 將湯圓放入滾水，以中細火煮至浮起熟透，盛起放碗內，淋上（4）即成。

4.富貴牡丹酥

01hr00min
製作難度：★★★★☆
份量：20件

牡丹酥炸起之後，猶如一朵盛開的花，代表富貴吉祥！

材料

水皮材料：麪粉266克、吉士粉/砂糖各38克、固體菜油76克、清水152克、紅/黃色食用色素各5克、金箔1片　

油皮材料：豬油304克、麪粉608克

餡料：蓮蓉1/2斤

做法

1. 先做水皮。將麪粉、吉士粉、砂糖和固體菜油拌勻搓一搓，逐少加入水搓成麪糰，然後分成兩份，一份加紅色色素，另一份加黃色色素。

2. 現做油皮。將豬油加麪粉搓勻。

3. 將黃色及紅色麪糰分別搓長，分別切成每件19克重一件；油皮搓長，切成每件11.4克；蓮蓉分成每件5克。

4. 將紅色皮及黃色皮分別包入油皮，分別輾平摺三摺再輾平。將紅色皮包黃色皮再包蓮蓉餡搓成球。另一粒用黃色皮包紅色皮再包入蓮蓉搓成球，然後放入雪櫃急凍5分鐘。

5. 在麪球上米字花，至皮開，但不要露餡，放入60℃油中炸至酥面盛開，慢慢加大火力炸至花開，約3至5分鐘，然後盛起瀝乾油，放上金箔即成。
