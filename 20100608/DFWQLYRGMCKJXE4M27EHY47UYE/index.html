煮題　15分鐘世界盃健康小食
杞子先浸酒

杞子不可用水洗，應用酒清洗。同時可用酒浸泡，然後才釀酒，可讓其吸收酒的香味。

靚沙葛皮光滑

沙葛應選外皮淺黃色及光滑為之靚，裏面的營養價值對腸胃有益。

蛋白塗杯邊　易蘸鹽

要令鹽粒均勻黏在杯口上，用蛋白塗杯邊後轉動杯，讓鹽牢牢黏滿杯口。捱夜吃點鹽，可以清熱下火。

波波小食拼盤

00hr15min
製作難度：★☆☆☆☆
份量：6人

睇波捱夜，一定要吃一些有熱量的東西，才會有精力。這個小食拼盤，除了多蔬菜，還有營養豐富的三文魚與牛油果，兩者都含不飽和脂肪酸，捱夜都不怕捱殘。

材料

水牛芝士丸6粒、煙三文魚2片、車厘茄6粒、青瓜1條、苜蓿芽適量、羅勒醬少許、牛油果1個、黑水欖6粒、亞麻籽1湯匙、乳酪75毫升、辣汁2茶匙、墨西哥原味粟米片1包

做法

1. 青瓜切厚片。黑水欖切碎。把煙三文魚片切開，包入半份水牛芝士丸，放在青瓜片上。

2. 把剩餘的水牛芝士丸切半，車厘茄切開，用竹籤把它們串起。

3. 將牛油果切開兩半，去核，挖肉再切成小粒，然後壓碎，加乳酪、黑水欖、辣汁和亞麻籽拌勻，食時伴墨西哥粟米片同吃。

4. 在碟上排上三文魚球及芝士番茄球，加上羅勒醬及苜蓿芽做裝飾即成。

酒香世界杯

00hr15min
製作難度：★☆☆☆☆
份量：6人

睇波鍾意飲啤酒，不過飲得多會有個大肚腩，營哥教你用水果配上疏肝明目的杞子酒，微醺之中睇波特別High呀！

材料

蜜瓜/西瓜各1/4個、火龍果2個、杞子1茶匙、杞子酒40毫升、橙2個（榨汁）、鹽1茶匙、蛋白1隻

做法

1. 火龍果切半後去皮，然後挖球。蜜瓜挖球、西瓜挖球，將水果丸拌勻。

2. 杞子放入杞子酒中浸泡10分鐘。

3. 橙汁加杞子酒拌勻。

4. 雞尾酒杯蘸上蛋白，轉動杯將之均勻地蘸上幼鹽。杯內放入水果丸，再加入杞子酒橙汁，加裝飾即成。

焗釀蘑菇素金錢雞

00hr15min
製作難度：★★☆☆☆
份量：6人

傳統的金錢雞，因為有肥豬肉所以超級肥膩。但改以素雞，配上叉燒醬醃過，幾可亂真的！

材料

蘑菇6粒、葵花籽油浸吞拿魚1罐、洋葱1/6個（切粒）、低脂芝士/意式香草/海鹽/子薑各少許、蒜茸1茶匙、素雞1條、沙葛1個

調味料：叉燒醬2湯匙、麻油少許

做法

1. 沙葛切去蒂，削去外皮，切半後吸出圓形，再切半厘米厚片。

2. 平底鑊加油，用中火炒香洋葱粒、蒜茸和意式香草。

3. 將（2）拌入已隔油的吞拿魚中，釀入已去蒂的蘑菇內，撒上海鹽及芝士碎揑實，放上焗盤。

4. 素雞切片。叉燒醬及麻油拌勻，加入素雞拌勻。用針穿上素雞、沙葛和子薑片，一共穿兩串，放上焗盤，掃上剩餘的醬汁，與蘑菇一起放入已預熱至220℃焗爐焗8分鐘即成。

營哥選用葵花籽油浸吞拿魚，較一般油浸及鹽水浸的更健康。

燒粟米沙律

00hr15min
製作難度：★★☆☆☆
份量：6人

粟米很高纖維，好易有飽肚感，睇波食粟米，好過食杯麪啦！

材料

粟米粒1盒（約400克）、馬蹄粒10粒（切碎）、黑椒碎1/6茶匙、低脂沙律醬5湯匙、苜蓿芽適量

做法

1. 粟米粒拌入2湯匙低脂沙律醬。

2. 馬蹄粒放入錫紙撻內，粟米粒、黑椒碎放面，再唧上剩餘低脂沙律醬，用火槍燒至金黃色，上面放苜蓿芽做裝飾即可。

一般的燒粟米都是用牛油，但牛油只得油脂，營哥選用低脂沙律醬，用橄欖油、蛋黃及醋製成，較有益健康。
