煮題　母親節愛心菜
牛蒡浸醋免變色

牛蒡連皮都有營養所以不必去皮，只用錫紙冲刷即可，另外因為含鐵質，切開後容易氧化，所以放入稀醋水浸，可以防止氧化變色。用剩的牛蒡，可用報包裹放入雪櫃蔬果箱，不易變乾。

南瓜延緩腸胃吸收

南瓜含豐富食用纖維，可延緩腸胃吸收，令血糖維持穩定，不易感到肚餓。

芝士放盒冷藏

用剩的芝士，可用保鮮紙包裹放入食物盒後，才放入雪櫃，以免芝士吸收其他食物的氣味。

牛蒡松子仁沙律

00hr40min
製作難度：★★☆☆☆
份量：4人

牛蒡絲爽脆清甜，而且含有豐富食用纖維，可以促進腸蠕動，改善便秘，更有降膽固醇功效，而且又有豐富鐵質，對媽媽來講好補，而且配上芝麻醬，味道酸甜可口，更帶陣陣麻香。

材料

牛蒡200克、松子仁適量、芝麻醬2湯匙、葱絲適量、醋適量、醬油1茶匙、糖1/2茶匙、麻油少許

做法

1. 用錫紙洗擦牛蒡後切幼絲，放入清水，加幾滴醋浸10分鐘。

2. 將牛蒡絲放入滾水，用大火灼3分鐘，撈起瀝乾。

3. 加入芝麻醬、醬油、麻油、糖拌勻上碟。

4. 最後灑上松子仁、葱絲，即成。

雜菜燜牛尾

01hr50min
製作難度：★★★☆☆
份量：4人

牛尾肉嫩鮮味，更含有膠原蛋白，食用後皮膚會又白又滑，比護膚品更有功效，而且用上多種蔬果來燜，香甜清新滲出馥郁酒香。

材料

牛尾500克、紅蘿蔔1個、洋葱（切件）1個、西芹1/4棵、番茄（切件）5個、蒜5瓣、月桂葉3片、茄膏1罐、紅酒400毫升、橄欖油適量、鹽適量、糖適量

做法

1. 紅蘿蔔切件；西芹切粗粒，備用。

2. 燒熱鍋下橄欖油，放入牛尾，用大火煎至轉啡色後，撈起備用。

3. 燒熱鍋放入橄欖油，加入洋葱、蒜、紅蘿蔔、西芹、茄膏用大火炒香。注入紅酒煮滾。

4. 拌入番茄、月桂葉、牛尾，轉小火燜煮至肉軟、收汁，加入鹽、糖調味，即可上碟。

買牛尾時，除請肉販代切，自己切亦很容易，只要在骨節之間落刀，就相當容易切開。

芝士朱古力磚

01hr00min
製作難度：★★☆☆☆
份量：4人

芝士可以補充鈣質，有助預防骨質疏鬆，一定啱媽媽，而加入可以釋放令人快樂的安多芬的朱古力，芝士略帶鹹香，帶出可可的氣味，而且質地軟滑，非常可口。

材料

60%朱古力100克、金文畢芝士（淨芝心）80克、可可粉適量

做法

1. 芝士去外皮後，置室溫待軟化。

2. 朱古力弄碎後，坐入熱水待溶化，將芝士加入，攪至半溶化。

3. 將牛油紙鋪入淺盤，倒入芝士朱古力漿，放入雪櫃冷藏至凝固。

4. 取出後切成方塊；將可可粉、朱古力塊放入保鮮袋搖勻，即成。

豆漿南瓜湯

00hr50min
製作難度：★★★☆☆
份量：4人

豆類製品含異黃酮素，可以補充女性荷爾蒙，相當有益滋潤，加入濃郁香甜南瓜湯，帶淡淡豆香，中和濃甜，味道平衡得剛剛好。

材料

南瓜（去皮去籽）200克、紅蘿蔔50克、雞湯300毫升、濃豆漿200毫升、番荽碎適量、鹽適量

做法

1. 南瓜切粒；紅蘿蔔切粒，備用。

2. 開大火，將雞湯煮滾，放入南瓜粒、紅蘿蔔煮軟。

3. 將湯汁放入攪拌器打爛後，倒回煲中。

4. 加入豆漿拌勻，最後撒少許鹽調味，上碗後淋上少許豆漿，灑上番荽，即成。
