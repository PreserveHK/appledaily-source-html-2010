煮題　即食麪四煮
煎麪餅用中火防黐鑊

煎Pizza伊麪餅時，要用中火燒熱鑊後才加油，這樣便不會黐鑊。

撈麪不用放水煮

炒撈麪只要用滾水浸軟，毋須開火煮，可避免麪條煮到過腍，然後再用半煎炸方法來做，效果會更香脆。

醃肉加湯粉

醃豬肉時可用即食麪湯粉，方便得來又原汁原味。

火腿番茄Pizza伊麪

00hr25min
製作難度：★★★☆☆
份量：4人

Pizza吃得多，但用伊麪當餅底又真係未試過，估唔到會咁香脆。

材料

即食伊麪2個、番茄醬2湯匙、火腿1片、番茄粒25克、粟米粒20克、青椒粒25克、菠蘿粒115克、車打芝士2片、油少許

做法

1. 開大火燒滾水，放入即食伊麪煮3分鐘，盛起過冷河後瀝乾水。

2. 將伊麪加入少許油拌勻。

3. 燒熱鑊下油，放入伊麪以中火煎3分鐘至乾身，放入圓焗盤上。

4. 將火腿切絲，鋪上伊麪餅上，再鋪上番茄粒、粟米粒、青椒粒及菠蘿粒。

5. 然後淋上番茄醬，將芝士撕碎鋪上餅面。

6. 將Pizza放入已預熱至200℃焗爐，焗10至15分鐘即成。

豉椒雞柳炒撈麪

00hr30min
製作難度：★★★☆☆
份量：2人

以豆豉醬炒過的雞肉好惹味，配上香脆的撈麪撈勻吃，味道剛好。

材料

去骨雞腿肉2件、紅椒/青椒各1/2個（切絲）、洋葱1/4個（切絲）、豆豉醬1湯匙、乾葱頭/蒜頭各1粒（切碎）、米酒1茶匙、生粉水適量、生抽1茶匙、即食撈麪1個

醃料：油2茶匙、生抽1茶匙、米酒1茶匙、鹽/胡椒粉各少許、粟粉1茶匙

做法

1. 將雞腿肉切件，加醃料拌勻醃15分鐘。

2. 將適量滾水倒入即食撈麪浸1至2分鐘，用筷子輕輕攪鬆，倒入筲箕隔乾水。

3. 將半杯油倒入熱鑊內、放入麪條，以中火半煎炸3至4分鐘至金黃酥脆後上碟。

4. 原鑊用中火炒香豆豉醬、乾葱碎、蒜茸、紅椒絲、青椒絲、洋葱絲，炒約1分鐘。

5. 然後加入雞件炒3至4分鐘至熟，下米酒、生抽及生粉水煮勻，鋪在麪上即可。

蟹肉燴麪

00hr25min
製作難度：★★★☆☆
份量：4人

這個蟹肉燴麪有點像酒樓的鴻圖窩麪，但Denice用了即食麪來做，煮至收水，味道更香濃。

材料

即食麪1個、韭黃適量（切段）、熟蟹肉80克、豬肉40克（剁碎）、冬菇數粒（切絲）

調味料：即食麪湯粉1/2包、雞湯1/4杯、水1/2杯、蠔油2湯匙、麻油少許

醃料：鹽少許、油2茶匙、生粉1茶匙、水2茶匙

做法

1. 開大火煲滾水放入即食麪煮3分鐘，盛起過冷河後瀝乾水。

2. 豬肉碎加油、鹽、生粉及水拌勻醃10分鐘。

3. 開中火燒熱鑊下油，將豬肉炒熟，再加熟蟹肉、冬菇絲及韭黃炒勻。

4. 然後注入雞湯、水、蠔油及麻油炒勻，待滾起時加入麪兜勻，再下即食麪湯粉拌勻，煮至汁料收乾便可上碟。

素食纖麥冷麪

00hr15min
製作難度：★★☆☆☆
份量：4人

這個冷麪是用了纖麥麪，麪質幼細而且不經油炸，最適合用來做冷麪。

材料

即食纖麥麪2個、青瓜絲/甘笋絲/芹菜絲/芽菜各1/4杯、牛油果1個、合桃碎/橄欖油各少許

醬料：芝麻醬2湯匙、蜜糖1湯匙、生抽1茶匙、浙醋2湯匙、麻油1湯匙、芥末醬1茶匙、胡椒粉少許

做法

1. 開大火燒滾水，放入纖麥麪煮3分鐘，盛起過冷河後瀝乾水，加少許橄欖油拌勻。

2. 將所有醬料拌勻成冷麪汁。

3. 牛油果去皮、起肉、切片。

4. 將麪條放上碟、鋪上青瓜絲、甘笋絲、芽菜、芹菜絲及牛油果，淋上冷麪汁、灑上少許合桃碎即成。
