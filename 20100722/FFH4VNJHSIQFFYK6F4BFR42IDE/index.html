生熟薏米用法不同

脹身的熟薏米能清熱疏肝，適合夏天煲水飲用；洋薏米即生薏米，較扁身兼細粒，有祛濕功效，但較寒凉，春天飲用較合時。

薏米水浸枸杞

01hr00min
製作難度：★★☆☆☆
份量：4人

夏日排汗多，必須經常補充水份，薏米有清熱功效，配上可清熱止渴、寧心健腦的枸杞，有助補充體力。

材料

熟薏米80克、水2公升、枸杞1斤、車厘茄8粒、薑3片

調味：海鹽少許

做法

1. 熟薏米洗淨，加水煲45分鐘成薏米水。

2. 枸杞摘下嫩菜葉，用滾水灼軟備用。

3. 車厘茄十字，用熱水浸一會，去皮備用。

4. 煮滾熟薏米水，放入薑片及車厘茄，加鹽調味。最後把薏米水連車厘茄淋在灼軟枸杞葉上，即成。

薏米有淡淡的甜米香，正好中和枸杞的苦味！

薄荷梳打

00hr10min
製作難度：★★☆☆☆
份量：2杯

冰凉清爽的薄荷梳打固然美味透心凉，但重點是最後加入的鹽，有助即時補充流汗時失去的鹽份。

材料

薄荷葉5片、士多啤梨4粒、青檸1個、梳打水1罐、冰粒少許

調味：鹽1/4茶匙

做法

1. 把薄荷葉洗淨，捲起切碎備用。

2. 士多啤梨洗淨去蒂，切碎備用。

3. 青檸榨汁，備用。

4. 杯中先放入冰粒，加入士多啤梨、薄荷葉和青檸汁，再加入梳打水和鹽即可。

夏天經常流汗，流失水份之餘也會流失人體需要的鹽份，急於補充鹽份，可飲一小杯鹽水。

烏梅蝦沙律

00hr30min
製作難度：★★★☆☆
份量：4人

蝦中的蛋白質可補充體力，配合帶有泰式風味的柚子橙沙律伴吃，份外醒神。

材料

海中蝦300克（灼熟）、柚子肉200克、橙1個、車厘茄6粒、西瓜900克

烏梅汁材料：烏梅2粒、泰式雞醬1湯匙、青檸1個、蜜糖1茶匙、辣汁/麻油各2茶匙

做法

1. 灼熟的海中蝦浸冰水令肉質保持彈性，去殼留尾備用。

2.橙去皮用小刀起肉。柚子肉撕開。車厘茄對切。青檸榨汁。

3. 烏梅浸水15分鐘後，去核，切茸。加入烏梅汁的其他材料再拌勻。

4. 把烏梅泰式汁拌勻柚子肉、橙肉、車厘茄和蝦肉。伴切件西瓜上碟。

沙律汁中加入生津止渴的烏梅，加上辣汁拌成鹹鹹辣辣的沙律，令人胃口大開，體力回復。

紫菜田雞瓜粒湯

01hr30min
製作難度：★★★★☆
份量：4-6人

別小看這個田雞瓜粒湯，田雞的營養價值高而且能補充體力，加上消暑的絲瓜和能補充碘質的紫菜，喝上一兩碗已經讓你精神百倍！

材料

田雞1斤（約4隻）、紫菜20克、中型絲瓜1條、西瓜皮180克、薑3片、水2.5公升

調味：海鹽少許

做法

1. 田雞洗淨，切開田雞髀再將身切件，汆水。煲滾水，將田雞放入，煲30分鐘。

2. 取出田雞髀，拆骨起肉。把骨和肉再放回鍋中煲15分鐘，隔走田雞骨。

3. 紫菜浸水備用，絲瓜和西瓜皮去綠皮，切粒。

4. 田雞湯煲滾，放入薑片、絲瓜和西瓜皮粒滾15分鐘，加入紫菜，最後加入調味即可飲用。

可用剪刀將浸泡後的紫菜剪碎，以便進食。
